Installation Instructions

  1. Download the ai-portal-extension.zip file
  2. Unzip it on your computer
  3. In Chrome, go to chrome://extensions/
  4. Enable "Developer mode" (toggle in the top-right corner)
  5. Click "Load unpacked" and select the unzipped folder
  6. The extension should now be active in your Chrome browser

How to Use

  • Type @servicename your question in the address bar (e.g., @chatgpt how do I make pasta)
  • Click on the extension icon to use the popup interface
  • Set up API keys in the options page for direct in-extension responses
